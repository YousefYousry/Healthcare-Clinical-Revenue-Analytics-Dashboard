# Healthcare-Clinical-Revenue-Analytics-Dashboard

This project is my Power BI project as part of the Digital Egypt Pioneers Initiative (DEPI) training.

This project focuses on clinical and revenue analysis, transforming raw healthcare data into interactive insights to monitor hospital performance and financial health.

## Project Overview
This Healthcare Dashboard provides a clear view of:
- **Total Revenue Performance** (Surpassing $1Bn)
- **Clinical Operations** (40.24K Admissions)
- **Medical Condition Analysis** (Cancer, Arthritis, etc.)
- **Insurance Provider Insights** (Cigna, Aetna, etc.)
- **Patient Demographics & Length of Stay**

## Key Insights
- **Total Revenue:** $1.02Bn
- **Total Admissions:** 40,245 patients
- **Top Revenue Driver:** Cancer department ($448M)
- **Average Length of Stay:** 18 days
- **Gender Balance:** 50.2% Male vs 49.8% Female

## Tools & Skills Used
- **Power BI Desktop**
- **DAX (Data Analysis Expressions)** for advanced measures
- **Power Query** for data transformation and cleaning
- **Data Modeling** (Star Schema)

## Dashboard Features
- **Interactive Tab Switcher** (Overview, Clinical, Financial)
- **Dynamic KPI Cards** with real-time filtering
- **Clean & Modern UI** with high-quality visualization
- **Strategic Objectives Tracking** for hospital management

## Project Workflow
1. **Data Cleaning:** Processing 55,000+ patient records using Power Query.
2. **Data Modeling:** Creating relationships between conditions, insurance, and billing.
3. **DAX Logic:** Building measures for revenue growth and operational averages.
4. **Dashboard Design:** Applying a minimalist, professional layout for medical data.
